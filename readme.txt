== Description ==
Displays all vendors from Interactive Advertising Bureau, which are used for CMP solutions as well. 
Useful if you want to display them on your data privacy page for example. Please be aware, that this is not a official solution and that the developer of this plugin ist not registered as CMP Provider, as this plugin is NOT a CMP solution.

== Installation ==

Just install it and customize it under Settings > IAB Vendor List.
To get the newest list please press the button: "Update List and language". You have to do this after installation as well.
